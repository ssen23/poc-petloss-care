### Step 2: App.jsx에 라우팅 추가

기존 App.jsx 파일을 수정합니다:

```jsx
import React from 'react';
import { BrowserRouter as Router, Route, Routes } from 'react-router-dom';
import NeulPoomLanding from './NeulPoomLanding';
import TrustPage from './TrustPage';
import ServicePage from './ServicePage';
import PartnersPage from './PartnersPage';
import ReviewsPage from './ReviewsPage';
import NeulPoomOnboarding from './site2/NeulPoomOnboarding';
// ... 기타 import

function App() {
  return (
    <Router>
      <Routes>
        {/* 홈 */}
        <Route path="/" element={<NeulPoomLanding />} />
        
        {/* ✨ 새로 추가된 페이지들 */}
        <Route path="/trust" element={<TrustPage />} />
        <Route path="/service" element={<ServicePage />} />
        <Route path="/partners" element={<PartnersPage />} />
        <Route path="/reviews" element={<ReviewsPage />} />
        
        {/* 기존 라우트들 */}
        <Route path="/neulpoom/onboarding" element={<NeulPoomOnboarding />} />
        {/* ... 기타 라우트 */}
      </Routes>
    </Router>
  );
}

export default App;
```

---

### Step 3: 개발 서버 재시작

```bash
npm run dev
```

## 📱 네비게이션 동작

### 1️⃣ 로고 클릭
```jsx
<button onClick={() => navigate('/')}>
  [로고] 늘품
</button>
```
→ **홈으로 이동**

### 2️⃣ 메뉴 클릭
```jsx
<button onClick={() => navigate('/trust')}>진단 신뢰성</button>
<button onClick={() => navigate('/service')}>서비스</button>
<button onClick={() => navigate('/partners')}>연계 서비스</button>
<button onClick={() => navigate('/reviews')}>후기</button>
```
→ **각 페이지로 이동**

### 3️⃣ 시작하기 버튼
```jsx
<button onClick={() => navigate('/neulpoom/onboarding')}>
  시작하기
</button>
```
→ **진단 시작 페이지로 이동 (기존 온보딩)**

## 🎨 디자인 일관성

모든 페이지가 동일한 디자인 시스템을 사용합니다:

### 색상
- 메인: Teal (500-600) & Emerald (500-600)
- 강조: 페이지별 그라데이션
  - 진단 신뢰성: Blue/Indigo
  - 서비스: Emerald/Lime
  - 연계: Purple/Pink

- 고정 위치 (fixed top-0)
- 스크롤 시 배경 불투명도 증가

### 타이포그래피
- 제목: text-4xl/5xl/6xl font-bold
- 본문: text-lg/xl text-gray-600
- 강조: font-semibold/bold

## 🐛 문제 해결

### 1. 페이지가 표시되지 않음
```bash
# lucide-react 아이콘 패키지 설치 확인
npm install lucide-react
```

### 2. 라우팅이 작동하지 않음
```bash
# react-router-dom 설치 확인
npm install react-router-dom
```

### 3. Tailwind CSS가 적용되지 않음
```bash
# tailwind.config.js에서 경로 확인
content: [
  "./index.html",
  "./src/**/*.{js,ts,jsx,tsx}",
],
```
